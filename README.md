The world is based on RC. Thus, *everything* can be encoded into RCNB.

RCNB is available in various languages: **Golang** [Javascript](https://github.com/rcnbapp/RCNB.js) [C](https://github.com/rcnbapp/librcnb) [PHP](https://github.com/rcnbapp/RCNB.php) [Pascal](https://github.com/rcnbapp/RCNB.pas) ([more..](https://github.com/rcnbapp/))
